Modul 2.3: Pose Project

Estimasi: 60 Menit
Praktikum

Pose Project menggunakan model PoseNet untuk mengekstraksi koordinat titik kunci tubuh (keypoints) seperti mata, telinga, bahu, siku, pergelangan tangan, pinggul, lutut, dan pergelangan kaki. Model ini sangat efektif untuk aplikasi kesehatan atau olahraga.

1. Tutorial Membuat Model Pose

Kita akan melatih model untuk mendeteksi gerakan Squat vs Stand.

  1. Persiapan: Pilih Pose Project di Teachable Machine.
  2. Pengumpulan Data:
    • Pastikan kamera menunjukkan seluruh tubuh (kepala hingga kaki).
    • Rekam kelas "Berdiri" dengan berbagai posisi.
    • Rekam kelas "Squat" (jongkok) dengan berbagai kedalaman.
  3. Magic of PoseNet: Perhatikan titik-titik warna pada tubuh Anda di browser. Teachable Machine tidak belajar dari gambar pixel, melainkan dari koordinat titik-titik tersebut.
  4. Training & Export: Train model dan export ke TFLite.
