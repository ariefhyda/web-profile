Teori & Praktikum

Modul ini memperkenalkan Google Teachable Machine — platform berbasis web dari Google yang memungkinkan siapa saja melatih model Machine Learning tanpa menulis kode. Anda cukup menyediakan data, melatih model secara visual, lalu mengekspor hasilnya untuk digunakan di aplikasi Flutter.

Apa itu Teachable Machine?

Teachable Machine (teachablemachine.withgoogle.com) adalah eksperimen web dari Google yang memungkinkan Anda membuat model ML secara interaktif melalui browser. Tidak perlu instalasi, tidak perlu coding, dan hasilnya bisa langsung diekspor ke format TFLite untuk Flutter.

Jenis Project di Teachable Machine

Teachable Machine mendukung tiga jenis project klasifikasi utama. Masing-masing dioptimalkan untuk kebutuhan yang berbeda:

Image Project

Klasifikasi objek melalui gambar. Menggunakan MobileNet V2 sebagai base model. Sangat cocok untuk deteksi benda, wajah, atau klasifikasi kategori visual.

Buka Modul Image →

Audio Project

Klasifikasi suara pendek berdurasi 1 detik. Menggunakan teknik Spectrogram untuk mengubah suara menjadi "gambar" sebelum diproses oleh model.

Buka Modul Audio →

Pose Project

Klasifikasi gerakan tubuh. Menggunakan model PoseNet untuk mendeteksi 17 titik kunci (keypoints) tubuh seperti siku, lutut, dan bahu.

Buka Modul Pose →

Cara Kerja Teachable Machine

Di balik antarmuka yang sederhana, Teachable Machine menggunakan teknik Transfer Learning — sebuah pendekatan di mana model yang sudah dilatih sebelumnya (pre-trained model) di-fine-tune dengan data baru Anda.

Base Model: MobileNet

Menggunakan arsitektur CNN ringan buatan Google yang sudah dilatih dengan jutaan gambar. Model ini sudah memahami fitur visual dasar secara komprehensif.

Transfer Learning

Hanya melatih lapisan terakhir dari MobileNet menggunakan data Anda. Proses ini jauh lebih cepat dan efisien dibandingkan melatih model dari nol.

Training di Browser

Semua proses kalkulasi training dijalankan langsung di browser menggunakan TensorFlow.js. Privasi terjamin karena data tidak dikirim ke server.

Mengapa Menggunakan Teachable Machine?

Keuntungan Utama
  • Tanpa Coding: Fokus pada kualitas data dan label daripada arsitektur model.
  • Eksperimen Cepat: Validasi ide Machine Learning Anda dalam hitungan menit.
  • Privasi Terjamin: Seluruh proses training terjadi di browser Anda, data tidak pernah meninggalkan perangkat.
  • Support TFLite: Mudah diintegrasikan ke aplikasi Android, iOS, maupun Flutter.
