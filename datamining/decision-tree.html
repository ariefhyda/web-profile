Modul 8: Decision Tree (Pohon Keputusan) & C4.5

Estimasi: 60 Menit
Teori & Interaktif

Decision Tree (Pohon Keputusan) adalah salah satu metode klasifikasi paling populer karena mudah diinterpretasikan oleh manusia. Algoritma ini memecah dataset menjadi himpunan bagian yang lebih kecil dan homogen berdasarkan aturan "Jika-Maka" (If-Then rules), membentuk struktur seperti pohon terbalik.

Struktur Decision Tree

• Root Node (Akar): Atribut paling membedakan data.
• Internal Node (Cabang): Atribut pengujian yang membagi data lebih lanjut.
• Leaf Node (Daun): Hasil akhir / keputusan / kelas target.

Algoritma C4.5: Membangun Pohon

Bagaimana kita menentukan atribut mana yang menjadi akar atau cabang? Algoritma C4.5 (pengembangan dari ID3) menggunakan konsep Entropy dan Gain Ratio dari Teori Informasi.

1. Entropy

Entropy mengukur tingkat ketidakteraturan (impurity) atau keacakan dari sebuah dataset. Jika data semuanya memiliki kelas yang sama, Entropy = 0 (murni). Jika terbagi rata 50/50, Entropy = 1.

Entropy(S) = ∑ -pi * log2(pi)

2. Information Gain

Gain mengukur seberapa banyak "informasi" atau pengurangan Entropy yang kita dapatkan dengan membagi data berdasarkan atribut tertentu.

Gain(S, A) = Entropy(S) - ∑ (|Sv| / |S|) * Entropy(Sv)

Catatan Penting: C4.5 menyempurnakan Information Gain dengan menggunakan Gain Ratio untuk menghindari bias terhadap atribut yang memiliki banyak nilai unik.

Implementasi di Python

Pustaka scikit-learn menggunakan algoritma CART (Classification and Regression Trees), yang mirip dengan C4.5 tetapi menghasilkan pohon biner dan menggunakan Gini Impurity secara default (meskipun bisa diatur ke Entropy).

decision_tree_sklearn.py 
from sklearn.tree import DecisionTreeClassifier
from sklearn.preprocessing import LabelEncoder
from sklearn import tree
import pandas as pd
import matplotlib.pyplot as plt

# 1. Dataset dan Preprocessing (sama seperti algoritma lain)
# Pastikan fitur dikonversi menjadi numerik
df = pd.read_csv('play_tennis.csv')
X = df.iloc[:, :-1].apply(LabelEncoder().fit_transform)
y = df.iloc[:, -1]

# 2. Inisialisasi Decision Tree (Menggunakan Entropy)
clf = DecisionTreeClassifier(criterion='entropy', random_state=42)

# 3. Latih Model
clf.fit(X, y)

# 4. Visualisasi Pohon (Menghasilkan grafik plot)
plt.figure(figsize=(12,8))
tree.plot_tree(clf, feature_names=X.columns, class_names=clf.classes_, filled=True)
plt.show()

# 5. Prediksi Data Baru
prediksi = clf.predict([[2, 0, 1, 0]]) # Misal: Sunny, Hot, High, Weak
print("Prediksi:", prediksi[0])
