Modul 5: Data Reduction dengan PCA

Estimasi: 45 Menit
Teori & Visualisasi

Semakin banyak fitur (dimensi) yang kita miliki dalam sebuah dataset, semakin kompleks algoritma akan memprosesnya. Terkadang, fitur-fitur tersebut sangat berkorelasi tinggi, redundant, atau membawa noise yang tidak bermanfaat. Masalah ini dikenal sebagai Curse of Dimensionality.

Oleh karena itu, kita butuh teknik Data Reduction. Salah satu yang paling fundamental dan ampuh adalah Principal Component Analysis (PCA).

Bagaimana PCA Bekerja?
  1. Standarisasi: menyamakan skala rentang variabel kontinu sehingga semua variabel berkontribusi secara adil.
  2. Matriks Kovarians (Covariance Matrix): Dihitung untuk mengidentifikasi tingkat korelasi atau keterkaitan antar variabel/fitur.
  3. Eigenvectors dan Eigenvalues: Dihitung dari matriks kovarians untuk menentukan arah (Principal Components) yang memiliki variansi / rentang sebaran terbesar.
  4. Proyeksi: Data dirotasi dan diproyeksikan ke vektor komponen utama. Ini memungkinkan kita membuang dimensi/fitur dengan variansi kecil tanpa kehilangan esensi pola datanya.

Implementasi di Python

Dengan Python dan Scikit-Learn, kita bisa secara praktis mereduksi dataset yang misalnya memiliki ratusan fitur/kolom menjadi 2 atau 3 fitur komponen utama (PC1, PC2) untuk kebutuhan visualisasi di Scatter Plot 2D atau 3D.

pca_example.py 
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import pandas as pd

# 1. Asumsikan 'df' adalah dataframe yang memiliki 30 dimensi/kolom fitur
X = df.drop('target', axis=1)

# 2. Standarisasi Data (Wajib dilakukan sebelum PCA)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 3. Inisialisasi PCA untuk reduksi menjadi 2 dimensi (2 Principal Components)
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)

# 4. Memasukkannya kembali ke DataFrame yang lebih ringkas
df_pca = pd.DataFrame(data=X_pca, columns=['PC1', 'PC2'])
print(df_pca.head())

# Melihat persentase variansi pola data yang sukses dipertahankan / dirangkum oleh PC
print("Explained Variance Ratio:", pca.explained_variance_ratio_)
