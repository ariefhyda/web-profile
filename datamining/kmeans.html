Modul 9: K-Means Clustering

Estimasi: 60 Menit
Teori & Interaktif Visual

Berbeda dengan algoritma yang telah kita pelajari sebelumnya (KNN, Naïve Bayes, Decision Tree) yang merupakan Supervised Learning (Data memiliki target/label), K-Means Clustering adalah algoritma Unsupervised Learning.

Tujuan dari K-Means adalah untuk menemukan kelompok (cluster) dalam data di mana label atau targetnya tidak diketahui. Algoritma ini akan mengelompokkan data yang memiliki karakteristik mirip ke dalam satu cluster yang sama. Huruf "K" merepresentasikan jumlah cluster yang ingin dibentuk.

Cara Kerja K-Means
  1. Tentukan jumlah cluster K.
  2. Pilih K titik secara acak dari dataset sebagai pusat cluster awal (Centroid).
  3. Assignment: Hitung jarak setiap data ke seluruh centroid, lalu kelompokkan data ke centroid yang paling dekat.
  4. Update: Hitung nilai rata-rata (mean) dari setiap kelompok untuk mencari titik pusat yang baru, lalu geser centroid ke titik tersebut.
  5. Ulangi langkah 3 dan 4 sampai centroid tidak lagi berpindah (Konvergen).

Metode Elbow (Mencari K Optimal)

Dalam praktiknya, kita seringkali tidak tahu berapa nilai K (jumlah cluster) yang paling tepat. Salah satu cara paling umum adalah menggunakan Elbow Method (Metode Siku).

Kita menjalankan K-Means berulang kali dengan nilai K yang berbeda (misal K=1 sampai 10), lalu menghitung WCSS (Within-Cluster Sum of Squares), yaitu total kuadrat jarak antara setiap titik dengan centroidnya. Saat diplot ke grafik, kita mencari "siku" (elbow)—titik di mana penurunan WCSS mulai melambat secara drastis.

Implementasi di Python

kmeans_clustering.py 
from sklearn.cluster import KMeans
import pandas as pd
import matplotlib.pyplot as plt

# 1. Load Data (Pastikan hanya fitur numerik, tidak ada label target!)
df = pd.read_csv('customer_data.csv')
X = df[['Annual Income', 'Spending Score']]

# 2. Elbow Method untuk mencari K optimal
wcss = []
for i in range(1, 11):
    kmeans = KMeans(n_clusters=i, init='k-means++', random_state=42)
    kmeans.fit(X)
    wcss.append(kmeans.inertia_) # inertia_ = nilai WCSS

# (Optional) Plot grafik WCSS vs K untuk melihat sikunya

# 3. Latih model dengan K yang terpilih (misal siku di K=5)
kmeans = KMeans(n_clusters=5, init='k-means++', random_state=42)
y_kmeans = kmeans.fit_predict(X)

# 4. Lihat Hasil Cluster
df['Cluster'] = y_kmeans
print(df.head())
