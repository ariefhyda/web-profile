Modul 7: Algoritma Naïve Bayes

Estimasi: 60 Menit
Teori & Interaktif

Naïve Bayes adalah algoritma klasifikasi machine learning yang didasarkan pada Teorema Bayes dengan asumsi "naif" bahwa semua fitur dalam dataset saling bebas (independen) satu sama lain.

Mengapa disebut "Naïve"?

Disebut naif karena algoritma ini mengasumsikan bahwa setiap atribut atau fitur pada data tidak saling mempengaruhi. Misalnya, jika kita memprediksi buah "Apel" berdasarkan fitur warna (merah), bentuk (bulat), dan ukuran (kecil). Naïve Bayes menganggap ketiga atribut ini bekerja secara terpisah dan independen dalam menentukan probabilitas buah tersebut adalah apel.

Teorema Bayes

Inti dari Naïve Bayes adalah Teorema Bayes, yang menyediakan cara untuk menghitung Probabilitas Posterior:

P(C | X) = P(X | C) × P(C)
P(X)
  • P(C | X) : Posterior Probability - Probabilitas kejadian kelas C jika fitur X terjadi.
  • P(X | C) : Likelihood - Probabilitas munculnya fitur X jika diketahui kelasnya adalah C.
  • P(C) : Prior Probability - Probabilitas munculnya kelas C sebelum melihat fitur apapun.
  • P(X) : Evidence - Probabilitas kemunculan fitur X secara umum.

Implementasi di Python

Dalam praktiknya, kita tidak perlu menghitung manual. Library scikit-learn telah menyediakan implementasi Naïve Bayes. Terdapat beberapa varian Naïve Bayes, seperti Gaussian (untuk data numerik kontinu), Multinomial (untuk data teks), dan Categorical.

naive_bayes_sklearn.py 
from sklearn.naive_bayes import GaussianNB
from sklearn.preprocessing import LabelEncoder
import pandas as pd

# 1. Siapkan Dataset
data = {
    'Cuaca': ['Sunny', 'Sunny', 'Overcast', 'Rainy'],
    'Suhu': ['Hot', 'Hot', 'Hot', 'Mild'],
    'Main': ['No', 'No', 'Yes', 'Yes']
}
df = pd.DataFrame(data)

# 2. Encode kategori teks menjadi angka
le = LabelEncoder()
X = df.iloc[:, :-1].apply(le.fit_transform)
y = le.fit_transform(df['Main'])

# 3. Inisialisasi dan Latih Model
model = GaussianNB()
model.fit(X, y)

# 4. Prediksi Data Baru
# Misal: Cuaca='Overcast', Suhu='Hot' (Perlu di-encode sesuai LabelEncoder)
prediksi = model.predict([[1, 0]])
print("Hasil Prediksi:", le.inverse_transform(prediksi)[0])
