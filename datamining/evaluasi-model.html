Modul 9: Evaluasi Model Klasifikasi

Estimasi: 45 Menit
Analisis & Metrik

Setelah kita melatih model klasifikasi (seperti KNN atau Naïve Bayes), pertanyaan selanjutnya adalah: Seberapa bagus model kita? Evaluasi model bukan hanya sekadar melihat akurasi, tetapi memahami performa model dalam mengenali setiap kelas secara mendalam.

1. Confusion Matrix

Confusion Matrix adalah tabel yang membandingkan nilai prediksi model dengan nilai sebenarnya (*Ground Truth*). Untuk klasifikasi biner, tabel ini memiliki 4 komponen utama:

ACTUAL CLASS
PREDICTED CLASS
3
True Positive (TP)
1
False Positive (FP)
2
False Negative (FN)
4
True Negative (TN)
Tips Menghafal (Logika 2 Kata)
1. Kata Kedua (P/N): Apa yang DIKATAKAN oleh model?
P = Model bilang Positif, N = Model bilang Negatif
2. Kata Pertama (T/F): Apakah ucapan model itu BENAR?
T = Ucapannya Benar, F = Ucapannya Salah
• TP (True Positive): Model bilang Positif, dan itu BENAR.
• TN (True Negative): Model bilang Negatif, dan itu BENAR.
• FP (False Positive): Model bilang Positif, tapi itu SALAH (Aslinya Negatif).
• FN (False Negative): Model bilang Negatif, tapi itu SALAH (Aslinya Positif).

Studi Kasus: Deteksi Virus

Total Data = 10. Nilai-nilai inilah yang kemudian dimasukkan ke dalam Confusion Matrix.

2. Metrik Utama

Dari empat nilai di atas, kita bisa menghitung berbagai metrik untuk menilai kualitas model:

Σ

Accuracy (Akurasi)

Seberapa sering model benar secara keseluruhan? (TP + TN) / Total

🎯

Precision (Presisi)

Dari semua yang diprediksi Positif, berapa yang benar-benar Positif? TP / (TP + FP)

🔍

Recall (Sensitivitas)

Dari semua data yang benar-benar Positif, berapa yang berhasil ditemukan model? TP / (TP + FN)

⚖️

F1-Score

Rata-rata harmonik antara Precision dan Recall. Berguna jika dataset tidak seimbang (*Imbalanced*).

Implementasi di Python

Menggunakan library scikit-learn, kita bisa mendapatkan laporan evaluasi lengkap hanya dengan satu perintah.

model_evaluation.py 
from sklearn.metrics import confusion_matrix, classification_report, accuracy_score
3. 
# Misal: y_true adalah data asli, y_pred adalah hasil prediksi model
y_true = [0, 1, 0, 1, 1, 0, 1, 1]
y_pred = [0, 1, 1, 1, 0, 0, 1, 1]
4. 
# 1. Confusion Matrix
cm = confusion_matrix(y_true, y_pred)
print("Confusion Matrix:\n", cm)
5. 
# 2. Accuracy Score
acc = accuracy_score(y_true, y_pred)
print(f"Akurasi: {acc * 100}%")
6. 
# 3. Laporan Klasifikasi Lengkap (Precision, Recall, F1)
report = classification_report(y_true, y_pred)
print("\nClassification Report:\n", report)
