Modul 10: Hierarchical Clustering

Estimasi: 45 Menit
Teori & Praktik

Selain K-Means, algoritma pengelompokan yang sangat populer adalah Hierarchical Clustering. Jika K-Means mencoba membagi data ke dalam sejumlah klaster secara langsung, Hierarchical Clustering membangun sebuah hierarki klaster tanpa perlu mengetahui jumlah kelompok di awal.

Ada dua pendekatan utama dalam algoritma ini:

  • Agglomerative (Bottom-Up): Setiap titik data dimulai sebagai klasternya sendiri. Kemudian, pasangan klaster yang paling mirip digabungkan langkah demi langkah sampai hanya tersisa satu klaster besar. Pendekatan ini yang paling sering digunakan.
  • Divisive (Top-Down): Kebalikan dari Agglomerative. Dimulai dengan semua titik data dalam satu klaster besar, lalu dipecah perlahan-lahan hingga setiap titik data menjadi klasternya sendiri.

Metode Jarak Antar Klaster (Linkage)

Dalam proses menggabungkan klaster (Agglomerative), kita harus menghitung jarak tidak hanya antar titik, tapi antar klaster. Ada beberapa kriteria penggabungan (Linkage Criteria) yang menentukan jarak ini:

Tipe-Tipe Linkage
  1. Single Linkage (Jarak Terdekat): Jarak antara dua klaster didefinisikan sebagai jarak minimum antara anggota satu klaster dengan anggota klaster lainnya. Cenderung membentuk klaster yang memanjang (chaining).
  2. Complete Linkage (Jarak Terjauh): Jarak didefinisikan sebagai jarak maksimum antara anggota klaster satu dan lainnya. Membentuk klaster yang lebih padat dan berbentuk bola.
  3. Average Linkage (Jarak Rata-rata): Menghitung rata-rata jarak antara seluruh pasangan titik data di kedua klaster. Merupakan kompromi antara Single dan Complete Linkage.
  4. Ward's Method: Menggabungkan dua klaster yang menghasilkan peningkatan varians terkecil (Within-Cluster Sum of Squares) secara keseluruhan. Sangat efektif untuk meminimalisasi varians dan sering menjadi pilihan default.

Dendrogram

Salah satu keunggulan terbesar Hierarchical Clustering adalah hasilnya dapat divisualisasikan dalam bentuk struktur pohon yang disebut Dendrogram. Dendrogram membantu kita melihat urutan penggabungan klaster secara vertikal dan mempermudah kita untuk memotong pohon tersebut pada level tertentu guna menentukan jumlah klaster yang optimal.

Implementasi di Python

Biasanya, kita menggunakan scipy untuk membuat visualisasi Dendrogram, dan scikit-learn untuk menerapkan model Agglomerative Clustering.

hierarchical_clustering.py 
import pandas as pd
import matplotlib.pyplot as plt
import scipy.cluster.hierarchy as sch
from sklearn.cluster import AgglomerativeClustering

# 1. Load Data
df = pd.read_csv('customer_data.csv')
X = df[['Annual Income', 'Spending Score']]

# 2. Membuat Dendrogram (untuk mencari K optimal)
plt.figure(figsize=(10, 7))
plt.title("Customer Dendrogram")
dendrogram = sch.dendrogram(sch.linkage(X, method='ward'))
plt.show()

# Berdasarkan dendrogram, kita bisa memilih threshold untuk memotong pohon.
# Misal, kita temukan K optimal = 5

# 3. Latih Model Agglomerative Clustering
hc = AgglomerativeClustering(n_clusters=5, metric='euclidean', linkage='ward')
y_hc = hc.fit_predict(X)

# 4. Tambahkan label klaster ke dataset
df['Cluster'] = y_hc
print(df.head())
