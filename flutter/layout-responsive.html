Modul 7: Layout & Responsive Design

Estimasi: 90 Menit
Teori & Praktikum

Aplikasi Flutter Anda akan dijalankan di berbagai ukuran layar: HP kecil 5", HP besar 6.7", tablet, bahkan desktop. Modul ini membahas cara membangun layout yang fleksibel dan responsif sehingga UI tetap rapi di ukuran apapun.

Buat Project Baru

Sesuai pendekatan mata kuliah ini — setiap modul = project baru. Buat project khusus untuk eksperimen layout:

Terminal
$ flutter create flutter_layout
$ cd flutter_layout
$ flutter run

Konsep Dasar Layout

Layout di Flutter mengikuti aturan constraint flow: parent memberikan batasan ukuran ke child, dan child mengembalikan ukuran yang ia inginkan. Memahami aturan ini akan menghindarkan Anda dari error "Vertical viewport was given unbounded height" yang sering muncul.

Constraint Flow (3 Aturan Emas)

1. Constraint mengalir turun (dari parent ke child).

2. Size mengalir naik (dari child ke parent).

3. Parent menentukan posisi child.

Expanded & Flexible

Saat menyusun widget dalam Row atau Column, ukurannya kadang harus dibagi proporsional. Di sinilah Expanded dan Flexible berperan. Mereka memberi tahu Flutter, "isi sisa ruang yang ada".

  1. Expanded — child wajib mengisi seluruh ruang yang dialokasikan.
  2. Flexible — child boleh lebih kecil dari ruang yang diberikan.
  3. flex — angka rasio pembagi ruang. Default 1. flex: 2 mengambil dua kali lipat dari flex: 1.
expanded.dart
Row(
  children: [
    Expanded(
      flex: 2, // Ambil 2/3 ruang
      child: Container(color: Colors.red, height: 100),
    ),
    Expanded(
      flex: 1, // Ambil 1/3 ruang
      child: Container(color: Colors.green, height: 100),
    ),
  ],
)

Stack & Positioned — Overlay Widget

Stack memungkinkan widget ditumpuk satu di atas yang lain (mirip CSS position: absolute). Gunakan Positioned untuk menempatkan child di koordinat tertentu.

stack.dart
Stack(
  alignment: Alignment.center,
  children: [
    Container(width: 200, height: 200, color: Colors.blue),
    const Icon(Icons.play_circle, size: 64, color: Colors.white),
    Positioned(
      top: 8,
      right: 8,
      child: Container(
        padding: const EdgeInsets.all(4),
        color: Colors.red,
        child: const Text('NEW', style: TextStyle(color: Colors.white)),
      ),
    ),
  ],
)

ListView — Daftar yang Bisa di-Scroll

Jika konten lebih panjang dari layar, gunakan ListView agar bisa di-scroll. Untuk daftar dinamis (misalnya 100 item), gunakan ListView.builder yang lazy — hanya render item yang terlihat.

listview.dart
// ListView statis
ListView(
  children: const [
    ListTile(leading: Icon(Icons.home), title: Text('Beranda')),
    ListTile(leading: Icon(Icons.person), title: Text('Profil')),
    ListTile(leading: Icon(Icons.settings), title: Text('Pengaturan')),
  ],
)

// ListView dinamis (lazy)
ListView.builder(
  itemCount: 100,
  itemBuilder: (context, index) {
    return ListTile(
      leading: CircleAvatar(child: Text('$index')),
      title: Text('Item ke-$index'),
      subtitle: const Text('Deskripsi singkat'),
      onTap: () => print('Klik item $index'),
    );
  },
)

GridView — Tata Letak Grid

Untuk menampilkan item dalam bentuk grid (mis. galeri foto, daftar produk), gunakan GridView.count (jumlah kolom tetap) atau GridView.builder (kombinasi grid dengan generator dinamis).

gridview.dart
GridView.count(
  crossAxisCount: 2,           // 2 kolom
  crossAxisSpacing: 12,
  mainAxisSpacing: 12,
  padding: const EdgeInsets.all(12),
  children: List.generate(10, (i) {
    return Container(
      decoration: BoxDecoration(
        color: Colors.indigo.shade100,
        borderRadius: BorderRadius.circular(12),
      ),
      child: Center(child: Text('Card $i')),
    );
  }),
)

MediaQuery — Mengetahui Ukuran Layar

MediaQuery memberi tahu kita ukuran layar perangkat saat aplikasi berjalan. Sangat berguna untuk membuat layout yang adaptif terhadap berbagai device.

mediaquery.dart
@override
Widget build(BuildContext context) {
  final size = MediaQuery.of(context).size;
  final isPortrait = MediaQuery.of(context).orientation == Orientation.portrait;

  return Scaffold(
    body: Center(
      child: Container(
        width: size.width * 0.8,   // 80% lebar layar
        height: size.height * 0.3,  // 30% tinggi layar
        color: isPortrait ? Colors.indigo : Colors.deepOrange,
        child: Center(
          child: Text('${size.width.toInt()} × ${size.height.toInt()}'),
        ),
      ),
    ),
  );
}

LayoutBuilder — Layout Adaptif

LayoutBuilder memberikan informasi constraint dari widget induknya. Cocok untuk menentukan tata letak berdasarkan ukuran parent (bukan ukuran layar). Ini adalah teknik andalan untuk membuat layout HP vs tablet yang berbeda.

layout_builder.dart
LayoutBuilder(
  builder: (context, constraints) {
    if (constraints.maxWidth < 600) {
      // Tampilan HP — 1 kolom
      return Column(children: items);
    } else {
      // Tampilan tablet/desktop — 2 kolom
      return Row(
        children: items.map((w) => Expanded(child: w)).toList(),
      );
    }
  },
)

SafeArea & SingleChildScrollView

Dua widget kecil tapi penting:

  1. SafeArea — menjaga konten Anda tidak tertutup oleh notch, status bar, atau navigation bar perangkat.
  2. SingleChildScrollView — membungkus widget biasa agar bisa di-scroll, cocok untuk form panjang.
safe_scroll.dart
Scaffold(
  body: SafeArea(
    child: SingleChildScrollView(
      padding: const EdgeInsets.all(16),
      child: Column(children: longList),
    ),
  ),
)

Mini Project: Galeri Foto Responsif

Mari kita gabungkan semuanya. Aplikasi ini menampilkan grid foto yang otomatis berubah jumlah kolom mengikuti lebar layar: 2 kolom di HP, 3 di tablet, 4 di desktop.

lib/main.dart
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.teal, useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('Galeri Foto')),
        body: LayoutBuilder(
          builder: (context, constraints) {
            int kolom;
            if (constraints.maxWidth < 600) {
              kolom = 2;
            } else if (constraints.maxWidth < 900) {
              kolom = 3;
            } else {
              kolom = 4;
            }

            return GridView.builder(
              padding: const EdgeInsets.all(12),
              gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
                crossAxisCount: kolom,
                crossAxisSpacing: 12,
                mainAxisSpacing: 12,
              ),
              itemCount: 20,
              itemBuilder: (context, index) {
                return ClipRRect(
                  borderRadius: BorderRadius.circular(12),
                  child: Image.network(
                    'https://picsum.photos/300?random=$index',
                    fit: BoxFit.cover,
                  ),
                );
              },
            );
          },
        ),
      ),
    );
  }
}
Uji Responsivitas

Cara cepat menguji layout di berbagai ukuran tanpa device fisik: jalankan flutter run -d chrome lalu drag jendela browser untuk merubah lebarnya. Anda akan melihat grid otomatis berubah jumlah kolomnya.

Tantangan Mahasiswa

1. Modifikasi galeri di atas: tambahkan Stack + Positioned untuk menampilkan teks "Foto #X" di pojok kanan-bawah setiap gambar.

2. Bungkus seluruh body dengan SafeArea agar konten tidak tertutup notch saat dijalankan di iPhone.

3. Tambahkan action button di AppBar yang ketika ditekan akan menampilkan SnackBar berisi ukuran layar saat ini (gunakan MediaQuery).
