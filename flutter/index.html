Modul 1: Pengenalan Flutter & Dart

Estimasi: 60 Menit
Teori & Pengantar

Selamat datang di modul pertama mata kuliah Mobile Programming. Pada perkuliahan ini kita akan mempelajari cara membangun aplikasi mobile cross-platform menggunakan Flutter, sebuah UI toolkit buatan Google yang memungkinkan Anda membuat aplikasi untuk Android dan iOS dari satu codebase tunggal.

Apa itu Flutter?

Flutter adalah framework open-source buatan Google untuk membangun aplikasi natively compiled yang berjalan di mobile (Android & iOS), web, dan desktop dari satu codebase tunggal. Flutter menggunakan bahasa pemrograman Dart dan mengadopsi pendekatan widget-based UI yang sangat modular.

Mengapa Flutter?

Flutter menawarkan Hot Reload untuk pengembangan super cepat, satu codebase untuk semua platform, serta performa mendekati native karena dikompilasi langsung ke kode mesin ARM. Komunitas Flutter juga berkembang sangat pesat sejak 2018.

Keunggulan Flutter

  1. Cross-Platform
    Satu codebase untuk Android, iOS, Web, dan Desktop. Hemat waktu dan biaya pengembangan secara signifikan.
  2. Hot Reload
    Perubahan kode langsung terlihat di emulator/device dalam hitungan milidetik tanpa kehilangan state aplikasi.
  3. Widget-Based Architecture
    Seluruh UI dibangun dari widget yang dapat disusun (composable), dikustomisasi, dan digunakan ulang dengan sangat fleksibel.
  4. Performa Native
    Flutter tidak menggunakan bridge seperti React Native. Kode Dart dikompilasi langsung ke ARM/x64 native code.
  5. Ekosistem Kaya
    Ribuan package tersedia di pub.dev untuk mempercepat development (state management, HTTP, database, dll).

Bahasa Pemrograman Dart

Dart adalah bahasa pemrograman modern yang dirancang oleh Google, dioptimalkan untuk membangun UI. Dart mendukung null safety, async/await, dan paradigma Object-Oriented Programming (OOP) secara penuh.

hello_dart.dart 
void main() {
  // Variabel dengan type inference
  var nama = 'Arif Hidayah';
  int umur = 28;
  double ipk = 3.93;
  bool isDosen = true;

  print('Halo, saya $nama');
  print('Umur: $umur, IPK: $ipk');
  print('Dosen: $isDosen');
}

Fitur Utama Dart

  1. Null Safety
    Sistem tipe yang aman dari null secara default. Variabel tidak boleh bernilai null kecuali dideklarasikan eksplisit dengan tanda ?.
  2. Async / Await
    Penanganan operasi asinkron (seperti pemanggilan API) yang elegan dan mudah dibaca, mirip JavaScript modern.
  3. OOP Lengkap
    Mendukung class, inheritance, mixin, abstract class, interface, dan generic types.
  4. Type Inference
    Dart bisa menebak tipe data secara otomatis menggunakan keyword var sehingga kode lebih ringkas.

Arsitektur Flutter

Flutter memiliki arsitektur berlapis (layered architecture) yang terdiri dari tiga lapisan utama:

Tiga Lapisan Arsitektur Flutter

1. Framework (Dart) — Layer teratas tempat Anda menulis kode. Berisi widget, gesture, animasi, dan painting API.

2. Engine (C/C++) — Mesin rendering Skia, text layout (menggunakan Impeller di Flutter 3.x+), dan Dart runtime.

3. Embedder (Platform-specific) — Layer terbawah yang berkomunikasi dengan OS (Android: Java/Kotlin, iOS: Objective-C/Swift).

Konsep Widget

Di Flutter, semuanya adalah widget. Sebuah tombol adalah widget. Sebuah padding adalah widget. Bahkan keseluruhan aplikasi Anda pun adalah sebuah widget. Widget tersusun dalam bentuk tree (pohon hierarki) yang dikenal sebagai Widget Tree.

Ada dua jenis widget fundamental:

  1. StatelessWidget
    Widget yang tidak memiliki state internal yang berubah. Cocok untuk UI statis seperti teks, ikon, atau gambar.
  2. StatefulWidget
    Widget yang memiliki state yang dapat berubah saat runtime. Cocok untuk form input, counter, animasi, dan komponen interaktif lainnya.
main.dart 
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        colorSchemeSeed: Colors.blue,
        useMaterial3: true,
      ),
      home: const Scaffold(
        body: Center(
          child: Text(
            'Halo Mahasiswa! 🎓',
            style: TextStyle(fontSize: 24),
          ),
        ),
      ),
    );
  }
}

Persiapan Lingkungan Pengembangan

Sebelum mulai ngoding Flutter, pastikan perangkat Anda telah memenuhi prasyarat berikut:

  1. Install Flutter SDK
    Download dari flutter.dev, extract, dan tambahkan ke PATH sistem.
  2. Install Android Studio
    Diperlukan untuk Android SDK, emulator, dan tools pendukung. Pastikan sudah install SDK API 33+.
  3. Editor: VS Code (Rekomendasi)
    Install extension Flutter dan Dart dari marketplace. VS Code lebih ringan dibanding Android Studio untuk coding sehari-hari.
  4. Verifikasi Instalasi
    Jalankan perintah flutter doctor di terminal untuk memastikan semua komponen siap digunakan.
Terminal 
# Cek instalasi Flutter
$ flutter doctor

# Buat project baru
$ flutter create nama_project

# Jalankan aplikasi
$ cd nama_project
$ flutter run
Tips Mahasiswa

Jika laptop Anda tidak kuat menjalankan emulator Android, gunakan perangkat fisik (HP Android) yang dihubungkan via USB dengan mode USB Debugging aktif. Atau, gunakan Zapp.run untuk latihan Dart & Flutter langsung di browser!
