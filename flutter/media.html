Modul 12: Pengelolaan Media (Image, Audio, File)

Estimasi: 100 Menit
Teori & Praktikum

Aplikasi nyata sering perlu bekerja dengan media: mengambil foto dari kamera, memilih gambar dari galeri, memutar audio, atau membaca/menulis file. Modul ini mengajarkan cara menggunakan tiga package penting: image_picker, file_picker, dan audioplayers, serta cara menyimpan file ke direktori aplikasi.

Buat Project Baru
Terminal
$ flutter create flutter_media
$ cd flutter_media

# Tambahkan tiga package
$ flutter pub add image_picker
$ flutter pub add file_picker
$ flutter pub add audioplayers
$ flutter pub add path_provider

Mengatur Permission

Mengakses kamera, galeri, dan file di perangkat memerlukan izin dari user. Setiap platform punya cara konfigurasinya sendiri.

Android — buka android/app/src/main/AndroidManifest.xml, tambahkan permission di dalam tag <manifest>:

AndroidManifest.xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
<uses-permission android:name="android.permission.READ_MEDIA_AUDIO" />
<uses-permission android:name="android.permission.INTERNET" />

iOS — buka ios/Runner/Info.plist, tambahkan deskripsi permission yang akan ditampilkan ke user:

Info.plist
<key>NSCameraUsageDescription</key>
<string>Aplikasi memerlukan kamera untuk mengambil foto.</string>
<key>NSPhotoLibraryUsageDescription</key>
<string>Aplikasi memerlukan akses galeri untuk memilih foto.</string>
<key>NSMicrophoneUsageDescription</key>
<string>Aplikasi memerlukan mikrofon untuk merekam audio.</string>
Permission Runtime di Android 13+

Sejak Android 13, READ_EXTERNAL_STORAGE diganti menjadi READ_MEDIA_IMAGES, READ_MEDIA_VIDEO, dan READ_MEDIA_AUDIO yang lebih granular. Daftarkan semuanya jika aplikasi Anda perlu mengakses media tertentu.

image_picker — Ambil Foto dari Kamera/Galeri

Package image_picker menyediakan dialog standar OS untuk memilih gambar dari galeri atau langsung memotret dari kamera. Hasil kembaliannya adalah XFile yang berisi path file.

image_picker_demo.dart
import 'dart:io';
import 'package:flutter/material.dart';
import 'package:image_picker/image_picker.dart';

class ImagePage extends StatefulWidget {
  const ImagePage({super.key});
  @override
  State<ImagePage> createState() => _ImagePageState();
}

class _ImagePageState extends State<ImagePage> {
  XFile? _foto;
  final _picker = ImagePicker();

  Future<void> _ambilDariKamera() async {
    final file = await _picker.pickImage(
      source: ImageSource.camera,
      imageQuality: 80,        // kompres ke 80% — hemat storage
      maxWidth: 1280,
    );
    if (file != null) setState(() => _foto = file);
  }

  Future<void> _ambilDariGaleri() async {
    final file = await _picker.pickImage(source: ImageSource.gallery);
    if (file != null) setState(() => _foto = file);
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Pilih Foto')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            _foto == null
              ? const Icon(Icons.image, size: 120, color: Colors.grey)
              : Image.file(File(_foto!.path), height: 240),
            const SizedBox(height: 24),
            Row(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                ElevatedButton.icon(
                  onPressed: _ambilDariKamera,
                  icon: const Icon(Icons.camera_alt),
                  label: const Text('Kamera'),
                ),
                const SizedBox(width: 12),
                OutlinedButton.icon(
                  onPressed: _ambilDariGaleri,
                  icon: const Icon(Icons.photo_library),
                  label: const Text('Galeri'),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}
XFile vs File

XFile adalah abstraksi cross-platform dari package image_picker — ia hanya menyimpan path. Untuk menampilkan di Image.file(), konversi dulu menjadi File dari dart:io seperti contoh di atas. Di Flutter Web, gunakan file.readAsBytes() + Image.memory() karena dart:io tidak tersedia.

Memilih Banyak Gambar Sekaligus

Untuk fitur seperti upload galeri, gunakan pickMultiImage yang langsung mengembalikan List<XFile>.

multi_image.dart
List<XFile> _fotos = [];

Future<void> _ambilBanyak() async {
  final list = await ImagePicker().pickMultiImage(imageQuality: 80);
  setState(() => _fotos = list);
}

// Tampilkan dengan GridView
GridView.builder(
  gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(crossAxisCount: 3),
  itemCount: _fotos.length,
  itemBuilder: (_, i) => Image.file(File(_fotos[i].path), fit: BoxFit.cover),
)

Menyimpan File ke Direktori Aplikasi

File yang dipilih lewat image_picker biasanya disimpan di direktori sementara dan bisa hilang sewaktu-waktu. Untuk menyimpan secara permanen, copy ke direktori aplikasi menggunakan path_provider.

simpan_file.dart
import 'dart:io';
import 'package:path_provider/path_provider.dart';
import 'package:path/path.dart' as p;

Future<File> simpanPermanen(XFile sumber) async {
  final dir = await getApplicationDocumentsDirectory();
  final namaFile = 'foto_${DateTime.now().millisecondsSinceEpoch}.jpg';
  final tujuan = File(p.join(dir.path, namaFile));
  return File(sumber.path).copy(tujuan.path);
}

// Pemakaian:
final xfile = await ImagePicker().pickImage(source: ImageSource.camera);
if (xfile != null) {
  final permanen = await simpanPermanen(xfile);
  print('Disimpan di: ${permanen.path}');
}
getApplicationDocumentsDirectory vs getTemporaryDirectory

Documents — persisten, bertahan sampai aplikasi di-uninstall. Untuk file yang harus disimpan (foto profil, dokumen user).

Temporary — bisa dibersihkan OS kapan saja. Untuk cache (thumbnail, download sementara).

file_picker — Pilih File Apa Saja

Untuk fitur seperti upload PDF, impor CSV, atau lampiran dokumen, gunakan file_picker. Package ini bisa memfilter berdasarkan tipe file.

file_picker_demo.dart
import 'package:file_picker/file_picker.dart';

// Pilih satu file PDF
Future<void> pilihPdf() async {
  final result = await FilePicker.platform.pickFiles(
    type: FileType.custom,
    allowedExtensions: ['pdf'],
  );
  if (result != null) {
    final file = result.files.single;
    print('Nama: ${file.name}');
    print('Ukuran: ${file.size} bytes');
    print('Path: ${file.path}');
  }
}

// Pilih banyak file gambar
Future<void> pilihGambar() async {
  final result = await FilePicker.platform.pickFiles(
    type: FileType.image,
    allowMultiple: true,
  );
  result?.files.forEach((f) => print(f.name));
}

// Pilih file apapun
final any = await FilePicker.platform.pickFiles(type: FileType.any);

audioplayers — Memutar Audio

Package audioplayers bisa memutar audio dari berbagai sumber: file asset (built-in), file lokal, atau URL streaming.

audio_demo.dart
import 'package:audioplayers/audioplayers.dart';

class AudioPage extends StatefulWidget {
  const AudioPage({super.key});
  @override
  State<AudioPage> createState() => _AudioPageState();
}

class _AudioPageState extends State<AudioPage> {
  final _player = AudioPlayer();
  bool _sedangMain = false;

  Future<void> _playUrl() async {
    await _player.play(UrlSource(
      'https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3',
    ));
    setState(() => _sedangMain = true);
  }

  Future<void> _playAsset() async {
    // Daftarkan dulu di pubspec.yaml: assets: - assets/sound.mp3
    await _player.play(AssetSource('sound.mp3'));
    setState(() => _sedangMain = true);
  }

  Future<void> _pause() async {
    await _player.pause();
    setState(() => _sedangMain = false);
  }

  @override
  void dispose() {
    _player.dispose(); // JANGAN lupa — supaya tidak memory leak
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Audio Player')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            IconButton(
              iconSize: 96,
              icon: Icon(_sedangMain ? Icons.pause_circle : Icons.play_circle),
              onPressed: _sedangMain ? _pause : _playUrl,
            ),
            const SizedBox(height: 12),
            Text(_sedangMain ? 'Memutar...' : 'Tap untuk play'),
          ],
        ),
      ),
    );
  }
}
Listen Progress Audio

Anda bisa subscribe ke stream untuk membuat seek bar: _player.onPositionChanged.listen((d) => ...) memberi posisi pemutaran, dan _player.onDurationChanged.listen((d) => ...) memberi total durasi. Gabungkan dengan widget Slider untuk UI player profesional.

Mini Project: Galeri Foto Pribadi

Aplikasi yang memungkinkan user menambah foto dari kamera/galeri, lalu menyimpannya ke direktori aplikasi sehingga tidak hilang setelah restart. Ganti lib/main.dart dengan kode berikut:

lib/main.dart
import 'dart:io';
import 'package:flutter/material.dart';
import 'package:image_picker/image_picker.dart';
import 'package:path_provider/path_provider.dart';
import 'package:path/path.dart' as p;

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});
  @override
  Widget build(BuildContext context) => MaterialApp(
    theme: ThemeData(colorSchemeSeed: Colors.pink, useMaterial3: true),
    home: const GalleryPage(),
  );
}

class GalleryPage extends StatefulWidget {
  const GalleryPage({super.key});
  @override
  State<GalleryPage> createState() => _GalleryPageState();
}

class _GalleryPageState extends State<GalleryPage> {
  List<File> _photos = [];

  @override
  void initState() { super.initState(); _loadPhotos(); }

  Future<void> _loadPhotos() async {
    final dir = await getApplicationDocumentsDirectory();
    final files = dir.listSync()
      .whereType<File>()
      .where((f) => f.path.endsWith('.jpg'))
      .toList();
    files.sort((a, b) => b.path.compareTo(a.path));
    setState(() => _photos = files);
  }

  Future<void> _tambahFoto(ImageSource src) async {
    final picked = await ImagePicker().pickImage(source: src, imageQuality: 80);
    if (picked == null) return;
    final dir = await getApplicationDocumentsDirectory();
    final nama = 'foto_${DateTime.now().millisecondsSinceEpoch}.jpg';
    await File(picked.path).copy(p.join(dir.path, nama));
    _loadPhotos();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Galeri (${_photos.length})')),
      body: _photos.isEmpty
        ? const Center(child: Text('Belum ada foto'))
        : GridView.builder(
            padding: const EdgeInsets.all(8),
            gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(
              crossAxisCount: 3, crossAxisSpacing: 8, mainAxisSpacing: 8),
            itemCount: _photos.length,
            itemBuilder: (_, i) => GestureDetector(
              onTap: () => showDialog(
                context: context,
                builder: (_) => Dialog(child: Image.file(_photos[i])),
              ),
              child: ClipRRect(
                borderRadius: BorderRadius.circular(8),
                child: Image.file(_photos[i], fit: BoxFit.cover),
              ),
            ),
          ),
      floatingActionButton: Row(
        mainAxisAlignment: MainAxisAlignment.end,
        children: [
          FloatingActionButton(
            heroTag: 'cam',
            onPressed: () => _tambahFoto(ImageSource.camera),
            child: const Icon(Icons.camera_alt),
          ),
          const SizedBox(width: 12),
          FloatingActionButton(
            heroTag: 'gal',
            onPressed: () => _tambahFoto(ImageSource.gallery),
            child: const Icon(Icons.photo_library),
          ),
        ],
      ),
    );
  }
}
Tantangan Mahasiswa

1. Tambahkan tombol hapus foto saat user long-press di item. Konfirmasi dengan AlertDialog sebelum menghapus.

2. Tampilkan tanggal & waktu foto diambil di overlay pojok bawah setiap thumbnail (ambil dari nama file timestamp).

3. Tambahkan halaman kedua: Audio Recorder sederhana. Petunjuk: gunakan package record untuk merekam, lalu putar dengan audioplayers.

4. Bonus: Tambahkan fitur share foto menggunakan package share_plus.
