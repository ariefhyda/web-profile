Modul 6: Widget Dasar Flutter

Estimasi: 90 Menit
Teori & Praktikum

Setelah memahami dasar Dart, sekarang waktunya menyentuh Flutter. Di modul ini Anda akan berkenalan dengan widget-widget fundamental yang akan Anda gunakan setiap hari: Scaffold, Text, Container, Image, Icon, Row, Column, dan berbagai tombol.

Buat Project Baru

Setiap modul Flutter di mata kuliah ini menggunakan project terpisah agar Anda berlatih dari awal dan tidak bercampur dengan materi sebelumnya. Buka terminal di folder kerja Anda dan jalankan:

Terminal
$ flutter create flutter_widget_dasar
$ cd flutter_widget_dasar
$ flutter run

Edit file lib/main.dart dan ganti seluruh isinya mengikuti contoh kode di modul ini. Setelah disimpan, tekan r di terminal untuk hot reload.

Konsep Widget Tree

Di Flutter, tampilan aplikasi dibangun seperti pohon hierarki. Sebuah widget bisa memiliki widget anak (child), dan widget anak bisa memiliki cucu, dan seterusnya. Pemahaman ini krusial — UI yang rumit sekalipun hanyalah susunan dari widget-widget sederhana.

widget_tree.txt
MaterialApp
 └── Scaffold
      ├── AppBar
      │    └── Text('Judul')
      └── body: Center
                └── Column
                     ├── Text('Halo')
                     └── ElevatedButton
                          └── Text('Klik')

Scaffold & AppBar

Scaffold menyediakan kerangka standar Material Design: AppBar di atas, body di tengah, FloatingActionButton, BottomNavigationBar, drawer, dan lain-lain. AppBar adalah title bar aplikasi.

lib/main.dart
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Widget Dasar',
      theme: ThemeData(colorSchemeSeed: Colors.indigo, useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('Halaman Pertama')),
        body: const Center(child: Text('Halo, mahasiswa! 👋')),
      ),
    );
  }
}

Text — Menampilkan Teks

Widget Text dipakai untuk menampilkan teks. Properti style menerima TextStyle yang berisi pengaturan font, warna, ukuran, dll.

contoh_text.dart
Text(
  'Selamat Belajar Flutter',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.indigo,
    letterSpacing: 1.2,
  ),
  textAlign: TextAlign.center,
)

Container — Kotak Serbaguna

Container adalah salah satu widget paling sering dipakai. Bayangkan seperti <div> di HTML. Container bisa mengatur width, height, padding, margin, warna latar, border, dan banyak lagi melalui properti decoration.

contoh_container.dart
Container(
  width: 200,
  height: 100,
  margin: const EdgeInsets.all(16),
  padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 8),
  decoration: BoxDecoration(
    color: Colors.indigo.shade100,
    borderRadius: BorderRadius.circular(12),
    border: Border.all(color: Colors.indigo, width: 2),
    boxShadow: [
      BoxShadow(color: Colors.black12, blurRadius: 8, offset: const Offset(0, 4)),
    ],
  ),
  child: const Center(child: Text('Sebuah kotak')),
)
EdgeInsets — Cara Menulis Padding/Margin

EdgeInsets.all(16) — semua sisi 16.
EdgeInsets.symmetric(horizontal: 16, vertical: 8) — kiri-kanan 16, atas-bawah 8.
EdgeInsets.only(left: 8, top: 4) — sisi tertentu saja.
EdgeInsets.fromLTRB(8, 16, 8, 0) — Left, Top, Right, Bottom.

Row & Column — Susunan Horizontal & Vertikal

Row menyusun widget anak secara horizontal, Column menyusun secara vertikal. Keduanya menerima properti children berupa list dari widget. mainAxisAlignment mengatur perataan di sumbu utama, crossAxisAlignment di sumbu silang.

row_column.dart
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.start,
  children: const [
    Text('Nama: Arif'),
    SizedBox(height: 8), // Spacer vertikal
    Text('NIM: 2021001'),
    SizedBox(height: 8),
    Row(
      children: [
        Icon(Icons.email, size: 16),
        SizedBox(width: 6),
        Text('arif@kampus.ac.id'),
      ],
    ),
  ],
)
SizedBox — Trik Spacer

SizedBox(height: 16) di dalam Column akan menjadi spacer vertikal, sedangkan SizedBox(width: 16) di dalam Row menjadi spacer horizontal. Lebih clean dibanding menambah padding pada setiap child.

Image — Menampilkan Gambar

Ada dua sumber gambar yang paling sering dipakai: dari internet (Image.network) atau dari folder asset lokal (Image.asset).

contoh_image.dart
// Dari URL
Image.network(
  'https://picsum.photos/200',
  width: 200,
  height: 200,
  fit: BoxFit.cover,
)

// Dari asset lokal (daftarkan dulu di pubspec.yaml)
Image.asset('assets/logo.png', width: 100)

// CircleAvatar untuk foto profil bulat
const CircleAvatar(
  radius: 40,
  backgroundImage: NetworkImage('https://picsum.photos/100'),
)
Mendaftarkan Asset Gambar

Untuk pakai Image.asset, buat folder assets/ di root project, simpan gambar di sana, lalu buka pubspec.yaml dan tambahkan:

pubspec.yaml
flutter:
  uses-material-design: true
  assets:
    - assets/
    - assets/logo.png

Lalu jalankan flutter pub get.

Button — Berbagai Jenis Tombol

Flutter punya beberapa jenis button bawaan dengan gaya berbeda. Semua tombol memiliki properti onPressed berupa callback function yang dijalankan saat ditekan. Jika onPressed: null, tombol otomatis ter-disable.

contoh_button.dart
// Tombol utama, paling menonjol
ElevatedButton(
  onPressed: () => print('Diklik!'),
  child: const Text('Simpan'),
)

// Tombol teks tanpa background
TextButton(
  onPressed: () {},
  child: const Text('Batal'),
)

// Tombol dengan border
OutlinedButton(
  onPressed: () {},
  child: const Text('Lihat Detail'),
)

// Tombol berbentuk ikon
IconButton(
  onPressed: () {},
  icon: const Icon(Icons.delete),
  color: Colors.red,
)

// Tombol dengan ikon di kiri
ElevatedButton.icon(
  onPressed: () {},
  icon: const Icon(Icons.send),
  label: const Text('Kirim'),
)

Mini Project: Profile Card

Gabungkan semua widget yang sudah dipelajari menjadi satu kartu profil sederhana. Ganti seluruh isi lib/main.dart Anda dengan kode berikut, lalu jalankan ulang dengan flutter run.

lib/main.dart
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.indigo, useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('Profile Card')),
        body: Center(
          child: Container(
            margin: const EdgeInsets.all(24),
            padding: const EdgeInsets.all(24),
            decoration: BoxDecoration(
              color: Colors.white,
              borderRadius: BorderRadius.circular(16),
              boxShadow: [
                BoxShadow(color: Colors.black12, blurRadius: 12, offset: const Offset(0, 4)),
              ],
            ),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                const CircleAvatar(
                  radius: 48,
                  backgroundImage: NetworkImage('https://picsum.photos/200'),
                ),
                const SizedBox(height: 16),
                const Text('Arif Hidayah',
                  style: TextStyle(fontSize: 20, fontWeight: FontWeight.bold)),
                const SizedBox(height: 4),
                Text('Dosen Mobile Programming',
                  style: TextStyle(color: Colors.grey.shade600)),
                const SizedBox(height: 16),
                Row(
                  mainAxisAlignment: MainAxisAlignment.center,
                  children: [
                    ElevatedButton.icon(
                      onPressed: () {},
                      icon: const Icon(Icons.email),
                      label: const Text('Email'),
                    ),
                    const SizedBox(width: 8),
                    OutlinedButton.icon(
                      onPressed: () {},
                      icon: const Icon(Icons.message),
                      label: const Text('Chat'),
                    ),
                  ],
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}
Tantangan Mahasiswa

Modifikasi Profile Card di atas:

1. Ganti foto, nama, dan jabatan dengan data diri Anda sendiri.

2. Tambahkan ikon dan label untuk jumlah followers, postingan, dan following seperti tampilan Instagram.

3. Tambahkan satu tombol ElevatedButton dengan label "Follow" di bagian bawah card.
