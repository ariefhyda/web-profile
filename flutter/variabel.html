Modul 3: Variabel & Tipe Data di Dart

Estimasi: 50 Menit
Teori & Praktikum

Pada modul ini kita akan mempelajari cara mendeklarasikan variabel, memahami tipe-tipe data bawaan Dart, serta konsep penting Null Safety yang menjadi ciri khas Dart modern.

Deklarasi Variabel

Dart menyediakan beberapa cara untuk mendeklarasikan variabel. Setiap cara memiliki karakteristik dan kegunaan berbeda:

variabel.dart 
void main() {
  // 1. var — tipe ditentukan otomatis (type inference)
  var nama = 'Arif';        // Dart tahu ini String
  var umur = 25;              // Dart tahu ini int

  // 2. Deklarasi eksplisit — tipe ditulis langsung
  String kota = 'Tegal';
  int semester = 5;
  double ipk = 3.85;
  bool aktif = true;

  // 3. final — hanya bisa di-assign sekali
  final nim = '2021001';
  // nim = '2021002'; // ❌ ERROR!

  // 4. const — konstanta compile-time
  const pi = 3.14159;
  const maxRetry = 3;
}
final vs const

final — nilainya di-assign saat runtime dan tidak bisa diubah setelahnya. Cocok untuk nilai yang baru diketahui saat program berjalan (contoh: final now = DateTime.now()).

const — nilainya harus sudah diketahui saat compile-time. Lebih optimal untuk nilai yang benar-benar konstan (contoh: const pi = 3.14).

Tipe Data Dasar (Built-in Types)

  1. int — Bilangan bulat tanpa desimal.
    int umur = 25;
  2. double — Bilangan desimal (floating-point 64-bit).
    double tinggi = 170.5;
  3. num — Supertype dari int dan double. Bisa menampung keduanya.
    num nilai = 99;
  4. String — Teks/karakter. Bisa menggunakan kutip tunggal atau ganda.
    String pesan = 'Halo Dunia';
  5. bool — Nilai logika: true atau false.
    bool lulus = true;
  6. dynamic — Tipe fleksibel, bisa berubah ke tipe apapun saat runtime.
    dynamic x = 10; x = 'teks';

Operasi String

Dart memiliki fitur String Interpolation yang sangat praktis menggunakan tanda $ untuk menyisipkan variabel langsung di dalam string:

string_ops.dart 
void main() {
  var nama = 'Arif';
  var umur = 25;

  // String Interpolation
  print('Nama saya $nama, umur $umur tahun');
  print('Tahun lahir: ${2026 - umur}');

  // Method String bawaan
  var teks = '  Hello Flutter  ';
  print(teks.trim());           // 'Hello Flutter'
  print(teks.toUpperCase());     // '  HELLO FLUTTER  '
  print(teks.contains('Flutter')); // true
  print(teks.replaceAll('Flutter', 'Dart'));

  // Multi-line String
  var paragraf = '''
Ini adalah string
yang bisa lebih dari
satu baris.
''';
}

Tipe Data Collection

Dart memiliki tiga struktur data koleksi utama:

collections.dart 
void main() {
  // ===== LIST (Array) =====
  List<String> matkul = ['Flutter', 'Laravel', 'Data Mining'];
  matkul.add('IoT');
  print(matkul[0]);       // 'Flutter'
  print(matkul.length);   // 4

  // ===== MAP (Key-Value) =====
  Map<String, dynamic> mahasiswa = {
    'nama': 'Budi',
    'nim': '2021001',
    'ipk': 3.75,
  };
  print(mahasiswa['nama']); // 'Budi'

  // ===== SET (Unik, tanpa duplikat) =====
  Set<int> angka = {1, 2, 3, 2, 1};
  print(angka); // {1, 2, 3} — duplikat otomatis dihapus
}

Null Safety

Sejak Dart 2.12, Null Safety menjadi fitur wajib. Secara default, semua variabel tidak boleh bernilai null. Ini mencegah error NullPointerException yang sangat umum di bahasa lain.

null_safety.dart 
void main() {
  // Non-nullable (default) — WAJIB punya nilai
  String nama = 'Arif';
  // String nama2; // ❌ ERROR: harus diinisialisasi

  // Nullable — boleh null (tambahkan ?)
  String? alamat;
  print(alamat);  // null (OK, karena nullable)

  // Null-aware operator (?. dan ??)
  print(alamat?.length);         // null (aman, tidak crash)
  print(alamat ?? 'Belum diisi'); // 'Belum diisi'

  // Late — inisialisasi ditunda
  late String token;
  token = fetchToken();
  print(token);
}

String fetchToken() => 'abc123xyz';
Rangkuman Operator Null Safety

? — Menandai variabel boleh null: String? x;

?. — Akses property hanya jika tidak null: x?.length

?? — Nilai default jika null: x ?? 'fallback'

! — Memaksa non-null (hati-hati!): x!.length

late — Menunda inisialisasi variabel non-nullable

Konversi Tipe Data

konversi.dart 
void main() {
  // String → int
  var angkaStr = '42';
  int angka = int.parse(angkaStr);

  // String → double
  double desimal = double.parse('3.14');

  // int/double → String
  String teks = angka.toString();
  String teks2 = desimal.toStringAsFixed(1); // '3.1'

  // int ↔ double
  double d = angka.toDouble();  // 42.0
  int i = desimal.toInt();       // 3 (terpotong)
}
