Flutter Intermediate

8 Modul • Intermediate Level
Teori & Praktikum

Selamat datang di course Flutter Intermediate. Course ini adalah lanjutan dari Flutter Fundamentals dan berfokus pada penggunaan GetX sebagai microframework all-in-one untuk state management, routing, dan dependency injection dalam pengembangan aplikasi Flutter profesional.

Prasyarat

Sebelum memulai course ini, pastikan Anda telah menyelesaikan Flutter Fundamentals (Modul 1-12). Anda harus sudah memahami Dart, Widget, State Management dasar (setState/Provider), dan REST API.

Daftar Modul

Berikut adalah modul-modul yang akan dipelajari dalam course ini:

  1. Review & Arsitektur Bersih
    Review konsep fundamental Flutter dan pengenalan prinsip Clean Architecture (Separation of Concerns, SOLID).
  2. Pengenalan GetX
    Mengenal GetX sebagai microframework: apa itu GetX, filosofi, dan perbandingan dengan Provider/Bloc.
  3. Arsitektur GetX
    Pola arsitektur GetX Pattern: Controller, Binding, View, dan struktur folder yang rapi.
  4. GetX CLI & Scaffolding
    Menggunakan GetX CLI untuk generate project, page, controller, dan model secara otomatis.
  5. State Management (GetX)
    Reactive state dengan Rx, Obx, dan GetBuilder. Perbandingan simple vs reactive state management.
  6. Routing & Middleware
    Named routes, parameter passing, middleware (auth guard), dan transition animations dengan GetX.
  7. Dependency Injection
    Get.put, Get.lazyPut, Get.find, dan Bindings untuk mengelola dependency antar modul.
  8. Final Project Intermediate
    Membangun aplikasi lengkap menggunakan arsitektur GetX: CRUD dengan REST API dan navigasi kompleks.
