Modul 6: Blade Templating

Estimasi: 50 Menit
Teori & Praktikum

Blade adalah template engine bawaan Laravel — ringan namun kuat. Tidak seperti template engine lain, Blade tidak melarang Anda memakai PHP murni. Setiap file Blade dikompilasi ke PHP biasa dan di-cache sampai berubah, jadi tidak ada overhead performa.

📝
Analogi: Surat Massal dengan Variabel

Blade ibarat mail merge di Word: Anda buat template "Halo {nama}, saldo Anda Rp {saldo}". Lalu Word mengisi {nama} dan {saldo} dengan data dari Excel untuk setiap penerima. Begitu juga Blade: template HTML dengan slot {{ $var }} yang diisi data dari controller.

Menampilkan Data dengan {{ }}

Variabel ditampilkan dengan kurung kurawal ganda. Laravel otomatis menjalankan htmlspecialchars() — mencegah serangan XSS (Cross-Site Scripting).

welcome.blade.php 
<!-- Menampilkan variabel (aman dari XSS) -->
<h1>Hello, {{ $name }}</h1>

<!-- Memanggil fungsi PHP -->
<p>Waktu sekarang: {{ date('H:i') }}</p>

<!-- Nilai default jika variabel kosong -->
<p>{{ $title ?? 'Judul Default' }}</p>

<!-- Output mentah TANPA escape (HATI-HATI XSS!) -->
<div>{!! $htmlContent !!}</div>

Struktur Kontrol: @if / @else

            
Blade punya direktif yang lebih bersih daripada <?php if ?> PHP murni:

            

                

                    

                    Blade
                    
                

                
<!-- IF / ELSEIF / ELSE -->
@if (count($records) === 1)
    Saya punya 1 data.
@elseif (count($records) > 1)
    Saya punya banyak data.
@else
    Saya tidak punya data.
@endif

<!-- UNLESS = kebalikan IF -->
@unless (Auth::check())
    Anda belum login.
@endunless

<!-- SWITCH -->
@switch($role)
    @case('admin')
        Dashboard Admin
        @break
    @case('user')
        Dashboard User
        @break
    @default
        Halaman Tamu
@endswitch

            


            
Direktif Auth Cepat

            
Beberapa shortcut yang sangat sering dipakai untuk autentikasi:

            

                

                    

                    Blade
                    
                

                
<!-- Tampil jika sudah login -->
@auth
    <a href="/dashboard">Dashboard</a>
@endauth

<!-- Tampil jika BELUM login -->
@guest
    <a href="/login">Login</a>
@endguest

<!-- Cek apakah variabel kosong -->
@empty($posts)
    <p>Belum ada post.</p>
@endempty

<!-- Cek role / izin (Gate) -->
@can('edit', $post)
    <button>Edit</button>
@endcan

            


            
Looping: @foreach & @forelse

            

                

                    

                    Blade
                    
                

                
<!-- foreach biasa -->
<ul>
@foreach ($users as $user)
    <li>{{ $user->name }}</li>
@endforeach
</ul>

<!-- forelse: gabungan foreach + handler kosong -->
@forelse ($users as $user)
    <li>{{ $user->name }}</li>
@empty
    <p>Belum ada user terdaftar.</p>
@endforelse

<!-- for, while, break, continue -->
@for ($i = 0; $i < 10; $i++)
    Iterasi: {{ $i }}
@endfor

            


            
Variabel $loop (Super Power Foreach)

            
Di dalam @foreach, Blade otomatis menyediakan variabel $loop dengan metadata berguna:


            

                
PropertyArti

                
                    
$loop->indexIterasi ke-berapa (mulai dari 0)

                    
$loop->iterationIterasi ke-berapa (mulai dari 1)

                    
$loop->firsttrue jika iterasi pertama

                    
$loop->lasttrue jika iterasi terakhir

                    
$loop->countTotal jumlah item

                    
$loop->remainingBerapa item tersisa

                    
$loop->parentAkses $loop dari foreach di luar (kalau nested)

                
            


            

                

                    

                    contoh $loop
                    
                

                
@foreach ($users as $user)
    <div class="{{ $loop->first ? 'highlight' : '' }}">
        {{ $loop->iteration }}. {{ $user->name }}
        @if ($loop->last) <em>(terakhir)</em> @endif
    </div>
@endforeach

            


            
Include — Memasukkan Partial

            
Untuk komponen yang berulang (navbar, footer, sidebar), gunakan @include:


            

                

                    

                    layout
                

                
<!-- File: resources/views/layouts/master.blade.php -->
@include('partials.navbar') <!-- → resources/views/partials/navbar.blade.php -->

<main>
    @include('partials.alert', ['type' => 'success', 'msg' => 'OK'])
@yield('content')
</main>

@include('partials.footer')

            


            

                

                    
                    Bahaya {!! !!} dan XSS
                

                
Gunakan {{ }} untuk semua output user. {!! !!} hanya untuk konten yang Anda benar-benar kontrol (misal HTML hasil sanitize dengan library seperti HTMLPurifier). Klik tombol XSS Demo di Playground di atas untuk lihat efeknya!
