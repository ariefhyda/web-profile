Modul 13: Membangun REST API

Estimasi: 45 Menit
Teori & Praktikum

Selain digunakan sebagai aplikasi web full-stack, Laravel juga merupakan framework yang sangat luar biasa untuk membangun backend RESTful API. API backend ini nantinya akan dikonsumsi oleh aplikasi Mobile (Android/iOS) atau framework Frontend SPA (seperti React, Vue, atau Angular).

Routing API

Semua route untuk API dideklarasikan di dalam file routes/api.php (bukan di web.php). Laravel secara otomatis akan menambahkan prefix /api pada URL tersebut.

routes/api.php 
use App\Http\Controllers\Api\ProductController;

// URL yang diakses nantinya: http://localhost:8000/api/products
Route::get('/products', [ProductController::class, 'index']);
Route::post('/products', [ProductController::class, 'store']);
Route::get('/products/{id}', [ProductController::class, 'show']);

// Atau gunakan apiResource (sama seperti resource, tapi menghilangkan route 'create' dan 'edit' yang menghasilkan view HTML)
Route::apiResource('products', ProductController::class);
Mengaktifkan API Routes di Laravel 11+

Mulai Laravel 11, file api.php tidak terinstal secara otomatis. Untuk mempublikasikannya, Anda perlu menjalankan perintah:
php artisan install:api

Mengembalikan JSON Response

Di Controller web, biasanya Anda me-return view return view('page');. Di API, Anda me-return collection Eloquent atau array, dan Laravel secara otomatis akan mengubahnya menjadi JSON.

app/Http/Controllers/Api/ProductController.php 
class ProductController extends Controller
{
    public function index()
    {
        // Mengembalikan koleksi model. Otomatis jadi JSON!
        return Product::all();
    }

    public function store(Request $request)
    {
        // Validasi
        $validated = $request->validate([
            'name' => 'required|string',
            'price' => 'required|integer'
        ]);

        $product = Product::create($validated);

        // Response custom menggunakan helper response()->json() dengan HTTP Status Code 201 (Created)
        return response()->json([
            'message' => 'Produk berhasil ditambahkan',
            'data' => $product
        ], 201);
    }
}

API Resources (Format Output yang Rapi)

Ketika membangun API berskala besar, mengembalikan Model Eloquent secara langsung bukanlah praktik terbaik karena akan mengekspos semua atribut (termasuk yang sensitif). Anda sebaiknya menggunakan Eloquent API Resources untuk menjadi "jembatan transformasi" dari database ke JSON.

Terminal 
php artisan make:resource ProductResource
app/Http/Resources/ProductResource.php 
public function toArray(Request $request): array
{
    // Mendefinisikan kolom apa saja yang boleh dilihat oleh API Consumer
    return [
        'id' => $this->id,
        'product_name' => $this->name, // Mengubah nama key output json
        'price_formatted' => 'Rp ' . number_format($this->price, 0, ',', '.'),
        'created_at' => $this->created_at->format('d-M-Y'),
    ];
}
app/Http/Controllers/Api/ProductController.php 
public function index()
{
    // Me-return data yang sudah di-wrap dan difilter oleh Resource
    return ProductResource::collection(Product::all());
}
