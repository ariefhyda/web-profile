Modul 11: Form dan Validasi

Estimasi: 50 Menit
Teori & Praktikum

Setiap data yang dikirim user (form login, registrasi, post baru, dll) tidak boleh dipercaya sampai divalidasi. Laravel menyediakan API validasi yang sangat ringkas — cukup tulis aturan, Laravel handle semuanya: cek, redirect kalau gagal, dan tampilkan pesan error.

🔍
Analogi: Petugas Imigrasi

Sebelum masuk negara, paspor Anda diperiksa: masih berlaku? foto sesuai? bukan paspor palsu? Jika salah satu gagal — Anda dipulangkan. Validasi Laravel berperan sama: setiap input form diperiksa berdasarkan "checklist" sebelum boleh disimpan ke database.

CSRF Protection — Wajib di Setiap Form

Setiap form dengan method POST/PUT/PATCH/DELETE wajib menyertakan token CSRF. Tanpa ini, request ditolak dengan error 419 Page Expired — Laravel mencegah serangan Cross-Site Request Forgery.

form.blade.php 
<form method="POST" action="/posts">
    @csrf  <!-- WAJIB! Otomatis inject <input type="hidden" name="_token"...> -->

    <input type="text" name="title">
    <button type="submit">Simpan</button>
</form>

<!-- Untuk method PUT/PATCH/DELETE (HTML tidak support langsung) -->
<form method="POST" action="/posts/1">
    @csrf
    @method('PUT') <!-- Override method ke PUT -->
    ...
</form>

Validasi di Controller

Method $request->validate() menerima array berisi aturan validasi. Jika gagal, Laravel otomatis redirect kembali ke form dengan pesan error.

PostController.php 
public function store(Request $request)
{
    // Jika gagal: auto-redirect back + flash $errors
    $validated = $request->validate([
        'title'        => 'required|unique:posts|max:255',
        'body'         => 'required|min:50',
        'email'        => 'required|email',
        'age'          => 'required|integer|between:18,99',
        'category_id'  => 'required|exists:categories,id',
        'thumbnail'    => 'nullable|image|max:2048', // 2MB max
    ]);

    // Hanya berjalan jika validasi sukses
    Post::create($validated);

    return redirect('/posts')->with('success', 'Post berhasil dibuat!');
}

Menampilkan Error di Blade

Saat validasi gagal, Laravel otomatis menyediakan variabel $errors di semua view. Juga ada helper old() untuk mengisi ulang input agar user tidak kehilangan apa yang sudah diketik.

create.blade.php 
<form method="POST" action="/posts">
    @csrf

    <!-- Tampilkan SEMUA error di atas form -->
    @if ($errors->any())
        <div class="alert alert-danger">
            <ul>
                @foreach ($errors->all() as $error)
                    <li>{{ $error }}</li>
                @endforeach
            </ul>
        </div>
    @endif

    <!-- Field Title -->
    <div>
        <label>Judul</label>
        <!-- old() mengisi ulang dengan nilai sebelumnya saat validasi gagal -->
        <input type="text" name="title" value="{{ old('title') }}"
               class="@error('title') is-invalid @enderror">

        <!-- @error: cek apakah field ini punya error -->
        @error('title')
            <p class="error">{{ $message }}</p>
        @enderror
    </div>

    <button type="submit">Simpan</button>
</form>

Form Request — Validasi Terpisah

Untuk validasi kompleks, pisahkan dari Controller ke Form Request class. Hasilnya: Controller lebih bersih, aturan lebih mudah dipakai ulang.

Terminal 
php artisan make:request StorePostRequest
app/Http/Requests/StorePostRequest.php 
class StorePostRequest extends FormRequest
{
    // Authorization: siapa yang boleh kirim form ini?
    public function authorize(): bool
    {
        return auth()->check(); // Hanya user login
    }

    // Aturan validasi
    public function rules(): array
    {
        return [
            'title' => 'required|unique:posts|max:255',
            'body'  => 'required|min:50',
        ];
    }

    // Custom error message
    public function messages(): array
    {
        return [
            'title.required' => 'Judul wajib diisi.',
            'title.unique'   => 'Judul sudah dipakai.',
            'body.min'       => 'Konten minimal 50 karakter.',
        ];
    }
}
PostController.php — bersih sekarang 
public function store(StorePostRequest $request) // type-hint FormRequest
{
    // Validasi sudah dijalankan otomatis sebelum masuk sini
    Post::create($request->validated());
    return redirect('/posts');
}
