Definisi Computer Vision

Computer Vision adalah bidang ilmu yang mempelajari bagaimana komputer dapat memperoleh pemahaman tingkat tinggi dari citra digital atau video. Tujuan utamanya adalah mengotomasi tugas-tugas yang dapat dilakukan oleh sistem visual manusia — mulai dari mengenali objek, memahami scene, hingga mengambil keputusan berdasarkan informasi visual.

Secara formal, Computer Vision berusaha menjembatani kesenjangan (semantic gap) antara representasi pixel mentah dalam sebuah citra dengan makna semantik yang dipahami manusia. Misalnya, sebuah foto 1920×1080 pixel hanyalah kumpulan angka bagi komputer, namun bagi manusia itu bisa berarti "seekor kucing sedang tidur di atas sofa".

Fun Fact

Computer Vision lahir secara formal pada tahun 1966, ketika Marvin Minsky di MIT memberikan tugas musim panas kepada mahasiswanya: "hubungkan sebuah kamera ke komputer dan buat komputer mendeskripsikan apa yang ia lihat." Tugas yang dianggap sederhana ini ternyata membutuhkan waktu lebih dari 50 tahun riset intensif untuk mendekati kemampuan manusia!

Sejarah Singkat Computer Vision

Perkembangan Computer Vision dapat diringkas menjadi beberapa era penting:

  1. 1960-an — Fondasi Awal
    MIT Summer Vision Project (1966) oleh Marvin Minsky. Larry Roberts memperkenalkan "Block World" — pengenalan objek 3D sederhana dari citra 2D.
  2. 1970-an — Representasi & Model
    David Marr membangun teori komputasional untuk vision: dari primal sketch → 2.5D sketch → 3D model. Munculnya konsep edge detection dan representasi visual berjenjang.
  3. 1980–1990-an — Feature-Based Era
    Munculnya algoritma klasik: Canny Edge Detector (1986), Harris Corner Detector (1988), dan SIFT (Scale-Invariant Feature Transform) oleh David Lowe (1999).
  4. 2000-an — Machine Learning Era
    Viola-Jones face detector (2001) memungkinkan deteksi wajah secara real-time. HOG + SVM untuk pedestrian detection. Kompetisi ImageNet dimulai (2009).
  5. 2012–Sekarang — Deep Learning Revolution
    AlexNet memenangkan ImageNet 2012 dengan margin besar menggunakan CNN. Lahirnya arsitektur ikonik: VGGNet, GoogLeNet, ResNet, YOLO, dan Transformer-based Vision models (ViT). Akurasi melampaui kemampuan manusia di banyak benchmark.

Pipeline Umum Computer Vision

Sebuah sistem Computer Vision tipikal mengikuti alur kerja (pipeline) berikut:

  1. Image Acquisition (Akuisisi Citra)
    Memperoleh citra dari kamera, sensor, atau dataset. Termasuk penanganan format (JPEG, PNG, RAW) dan resolusi.
  2. Preprocessing
    Menyiapkan citra agar optimal untuk analisis: resize, normalisasi warna, noise reduction, konversi color space (RGB → Grayscale, HSV).
  3. Feature Extraction
    Mengekstrak fitur-fitur penting dari citra: tepi (edges), sudut (corners), tekstur, histogram warna, atau fitur deep learning dari CNN.
  4. Model / Inference
    Menggunakan algoritma atau model terlatih untuk menginterpretasi fitur: klasifikasi (ini gambar apa?), deteksi (di mana objeknya?), segmentasi (piksel mana milik objek apa?).
  5. Post-processing & Output
    Menyempurnakan hasil: Non-Maximum Suppression (NMS), thresholding, dan menyajikan output (bounding box, mask, label, atau aksi).

Aplikasi Computer Vision di Dunia Nyata

Computer Vision kini telah diterapkan secara luas dalam berbagai industri. Berikut beberapa contoh aplikasi paling impactful:

  1. Self-Driving Cars (Mobil Otonom)
    Tesla, Waymo, dan perusahaan lain menggunakan CV untuk lane detection, traffic sign recognition, pedestrian detection, dan navigasi real-time.
  2. Medical Imaging
    Analisis X-ray, CT-scan, dan MRI untuk mendeteksi tumor, penyakit paru-paru, retinopati diabetik, dan patologi lainnya dengan akurasi mendekati dokter spesialis.
  3. Face Recognition & Biometrics
    Sistem keamanan, unlock smartphone (Face ID), verifikasi identitas di bandara, dan sistem absensi berbasis wajah.
  4. Augmented Reality (AR)
    Filter Instagram/Snapchat, IKEA Place (preview furnitur di rumah), dan game seperti Pokémon Go menggunakan CV untuk memahami dan berinteraksi dengan lingkungan visual.
  5. Quality Control (Manufaktur)
    Inspeksi visual otomatis pada lini produksi untuk mendeteksi cacat produk, mengukur dimensi, dan memverifikasi perakitan komponen.
  6. OCR & Document Understanding
    Konversi dokumen cetak/tulisan tangan menjadi teks digital, ekstrasi data dari invoice/receipt, dan pengenalan plat nomor kendaraan (ANPR).

Apa yang Akan Dipelajari?

Kursus ini dirancang dengan pendekatan 70% praktikum dan 30% teori, mengikuti alur pembelajaran berikut:

  1. Representasi Citra Digital
    Memahami bagaimana citra direpresentasikan sebagai array numerik: pixel, channel warna (RGB, Grayscale, HSV), resolusi, dan bit depth.
  2. Operasi Dasar & Image Processing
    Transformasi geometris, filtering spasial, konvolusi, dan teknik peningkatan kualitas citra (histogram equalization, contrast stretching).
  3. Edge Detection & Feature Extraction
    Algoritma deteksi tepi (Sobel, Canny), corner detection (Harris), dan deskriptor fitur modern (SIFT, HOG).
  4. Deep Learning untuk Computer Vision
    Arsitektur CNN, transfer learning, dan implementasi menggunakan PyTorch/TensorFlow untuk klasifikasi citra.
  5. Object Detection & Segmentation
    YOLO untuk real-time object detection, semantic/instance segmentation, dan penerapan di berbagai domain.
