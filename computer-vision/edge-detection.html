Modul 5: Deteksi Tepi (Edge Detection)

Estimasi: 75 Menit
Teori, Simulasi & Praktikum

Edge Detection (deteksi tepi) merupakan salah satu operasi paling fundamental dalam Computer Vision. Tepi dalam citra merepresentasikan batas antara dua region yang berbeda — perubahan tajam pada intensitas pixel yang menandakan kontur objek, tekstur, atau bayangan.

Mengapa Edge Detection Penting?

Edge detection berfungsi sebagai langkah pra-proses (preprocessing) yang krusial karena:

  1. Reduksi Data
    Mengubah citra berukuran besar menjadi representasi ringkas berupa garis-garis tepi, mengurangi data hingga 90% tanpa kehilangan informasi struktural penting.
  2. Segmentasi Objek
    Tepi membantu memisahkan objek dari latar belakang, menjadi dasar untuk proses segmentasi dan identifikasi kontur.
  3. Feature Extraction
    Banyak deskriptor fitur (HOG, SIFT) memanfaatkan informasi gradient/tepi sebagai komponen utama.
  4. Scene Understanding
    Peta tepi memberikan informasi struktural tentang bentuk, batas, dan hubungan geometris antar objek.

Konsep Matematika: Gradien Citra

Tepi didefinisikan sebagai lokasi di mana terjadi perubahan drastis pada intensitas pixel. Secara matematis, ini dihitung menggunakan turunan pertama (gradient) dari fungsi intensitas citra.

Untuk citra 2D f(x,y), gradien dihitung pada dua arah:

gradient_formula.txt 
# Gradien pada arah X (horizontal)
Gx = ∂f/∂x ≈ f(x+1, y) − f(x−1, y)

# Gradien pada arah Y (vertikal)
Gy = ∂f/∂y ≈ f(x, y+1) − f(x, y−1)

# Magnitude (kekuatan tepi)
|G| = √(Gx² + Gy²)

# Arah tepi
θ = arctan(Gy / Gx)

Algoritma Edge Detection

1. Operator Sobel

Menggunakan kernel konvolusi 3×3 untuk menghitung gradien arah X dan Y secara terpisah, dengan memberikan bobot lebih pada pixel terdekat:

sobel_kernels.txt 
# Kernel Sobel X (deteksi tepi vertikal)
Kx = [[-1,  0,  1],
      [-2,  0,  2],
      [-1,  0,  1]]

# Kernel Sobel Y (deteksi tepi horizontal)
Ky = [[-1, -2, -1],
      [ 0,  0,  0],
      [ 1,  2,  1]]

2. Canny Edge Detector

Dikembangkan oleh John Canny (1986), merupakan detektor tepi paling populer dan dianggap sebagai gold standard. Canny memiliki 4 tahapan:

  1. Gaussian Blur
    Menghaluskan citra untuk mengurangi noise. Ukuran kernel Gaussian memengaruhi tingkat smoothing.
  2. Gradient Calculation
    Menghitung magnitude dan arah gradien menggunakan operator Sobel.
  3. Non-Maximum Suppression (NMS)
    Menipiskan tepi menjadi lebar 1 pixel dengan menekan nilai non-maksimum di sepanjang arah gradien.
  4. Double Thresholding & Edge Tracking
    Menerapkan dua threshold (tinggi dan rendah) untuk mengkategorikan tepi menjadi strong, weak, dan non-edge. Kemudian melakukan hysteresis — weak edge yang terhubung dengan strong edge dipertahankan.

Praktikum: Edge Detection dengan OpenCV

edge_detection.py 
import cv2
import numpy as np
import matplotlib.pyplot as plt

# Membaca citra dan konversi ke grayscale
img = cv2.imread('sample.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 1. Sobel Edge Detection
sobel_x = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=3)
sobel_y = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=3)
sobel_mag = np.sqrt(sobel_x**2 + sobel_y**2)
sobel_mag = np.uint8(sobel_mag / sobel_mag.max() * 255)

# 2. Canny Edge Detection
edges_canny = cv2.Canny(gray, threshold1=50, threshold2=150)

# 3. Laplacian Edge Detection
laplacian = cv2.Laplacian(gray, cv2.CV_64F)
laplacian = np.uint8(np.absolute(laplacian))

# Visualisasi Perbandingan
fig, axes = plt.subplots(2, 2, figsize=(12, 10))
axes[0,0].imshow(gray, cmap='gray'); axes[0,0].set_title('Original (Grayscale)')
axes[0,1].imshow(sobel_mag, cmap='gray'); axes[0,1].set_title('Sobel Magnitude')
axes[1,0].imshow(edges_canny, cmap='gray'); axes[1,0].set_title('Canny Edge')
axes[1,1].imshow(laplacian, cmap='gray'); axes[1,1].set_title('Laplacian')
for ax in axes.flat: ax.axis('off')
plt.tight_layout(); plt.show()
